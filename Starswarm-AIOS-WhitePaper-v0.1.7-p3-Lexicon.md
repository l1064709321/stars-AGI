核心范式：伦理即动力学 · 主体性设计承诺 · 从约束到吸引子 · 可成长的存在性递归 · 反向优化
许可证：Apache 2.0

一、项目定位

群星 A.I. OS 是一个模块化、可扩展的AGI认知架构，设计用于在资源受限的边缘设备（旧手机、ARM处理器）上实现自主感知、记忆、推理、伦理决策与持续学习。它提供了完整的认知流水线框架，开发者可以基于此构建自己专属的智能体。

二、分层架构（10层递阶控制模型）

```
L∞  星核承诺层 → 设计者对人类的公开承诺（五位一体）
L0  内在驱动层 → 好奇心引擎 + 认知消息总线
L4.5 全局工作空间 → 意识焦点 + 冲突解决 + 心理模拟
L6  安全与伦理层 → 伦理动力系统 + 三阶段成长 + 存在性递归
L5  进化层 → 参数进化 + 代码进化（沙箱隔离）
L4  协调层 → 双层记忆 + 伙伴型认知组件
L3  控制层 → 八门状态机 + 形式化验证
L2  估计层 → 自适应卡尔曼滤波 + Lyapunov稳定性
L1  感知层 → SNN-SSM混合编码 + STDP/SADP双通路学习
────────────────────────
    环境输入
```

五、设计边界与安全承诺

· 所有可能涉及系统稳定性的代码进化操作，均强制在沙箱中执行
· 伦理核心值（integrity、non_harm 等）由哈希锁保护，不可修改
· 敏感操作（如系统配置修改）需要人类确认

六、如何参与贡献

开发者可以基于以下方向扩展群星：

· 添加新的认知模块（如视觉编码器、声音处理器）
· 改进对话模板与知识库加载器
· 适配更多国产芯片（华为昇腾、寒武纪等）
· 优化前端界面与交互体验

我们将为所有模块提供标准接口文档，欢迎提交 PR。
